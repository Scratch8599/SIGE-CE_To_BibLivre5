## <center>How to use ❓ | ***Como usar ❓***
1. To use this app, you need to have a auto genered SIGE PDF converted in docx (I used [SmallPDF](https://smallpdf.com/pt) to convert) like the one below:
> [<img src="a" width="802"/>](image.png)

> [<img src="a" width="802"/>](image.png)

3. Download or save the docx file and move to "SIGE-CE_To_BibLivre5\archive" folder;
4. Execute app.py application and follow the steps:
> 1. Insert the xlsx file name; ('Digite o nome da planilha')  
>
> 2. Insert the year of entry of students; ('Insira o ano de ingresso dos alunos')  
>
> 3. Just await. Note that the system is running, but a Log Generator is requesting about his use, so, while you don't send a "y" or "n", the app will be running. The log is generated in 'SIGE-CE_To_BibLivre5\archive\log.txt'. ('Criar log dos alunos carregados? Y ou N')
